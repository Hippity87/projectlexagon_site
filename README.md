# ProjectLexagon.fi — Main Site

The primary landing site for the **ProjectLexagon** ecosystem.  
Acts as a hub to project pages and demos, including the Blazor and React subdomains.

---

## Overview

This repository contains the **main website** (static content + simple JS/CSS) shown at **projectlexagon.fi**.  
It links out to sub-projects that live in their own repositories/subdomains:

- **QuantumConnect (Rework)** — Blazor WebAssembly game UI  
  ↳ `quantum.projectlexagon.fi` (repo: `QuantumConnect_rework`)
- **React Frontend experiments** — Vite + Zustand + Router + Tailwind  
  ↳ `react.projectlexagon.fi` (repo: `loppuharkkafrontend`)

This repo focuses on the landing experience, navigation, and branding.

---

## Features

- Clean landing page with quick access to subprojects.
- Lightweight layout, static assets, and basic SEO metadata.
- Proof of Concept to deliver a site without frameworks.
